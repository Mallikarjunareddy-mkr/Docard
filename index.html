How to use Docard

  • Open Local Storage to manage files saved in your browser.
  • Use the FAB (+) to create folders or upload files.
  • Select items (checkbox) to show edit actions (rename, delete, download, details).
  • Files persist in localStorage so they survive page reloads.

Settings
